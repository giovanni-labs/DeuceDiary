# Deuce Diary iOS v1.0 — Product Requirements Document
_Generated by Alex (PM) via qwen2.5:7b local model — 2026-02-19_

## 1. Problem Statement
People in friend groups want a fun, low-friction way to share and react to each other's bathroom visits in real time. Current social apps take themselves too seriously. Deuce Diary fills the gap: a dedicated, irreverent logging and sharing tool for the throne room.

## 2. Target User
- Young adults (18–35) in established friend groups
- People who enjoy light social humor and group banter
- Active smartphone users comfortable with social apps

## 3. Core Features for v1 (Must-Haves)
1. **Deuce Logging** — one-tap log with optional location + thought
2. **Real-Time Group Feed** — live updates when friends log via WebSocket
3. **Friend Group Management** — create groups, invite via link, manage members
4. **Emoji Reactions** — react to friends' entries instantly
5. **Push Notifications** — notified when a friend drops a deuce
6. **User Profile** — username, photo, deuce stats, personal records

## 4. Out of Scope for v1
- Direct messaging / group chat
- Nearby restroom finder
- External social media integration
- Public/discovery feeds
- Badges and achievements (v2)
- Video/media in entries
